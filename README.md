# 📚 Repositório de Estudos em Programação

Este repositório reúne projetos, exercícios e práticas desenvolvidos durante meus estudos em programação e desenvolvimento de software.

O objetivo deste espaço é registrar minha evolução técnica, aplicar conceitos aprendidos em aula e praticar lógica de programação, estrutura de dados e desenvolvimento de aplicações.

---

## 👨‍💻 Sobre mim

Estudante de **Engenharia de Software pela UNINTER**, com foco em desenvolvimento e aprimoramento de soluções voltadas para **automação e processamento de dados (back-end)**.

Busco constantemente aprimorar minhas habilidades em programação através de projetos práticos e estudos contínuos.
